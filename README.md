## Tech Stack

- UI: Jetpack Compose, Material3
- Architecture: MVVM, StateFlow
- Networking: Retrofit, Gson
- Dependency Injection: Koin
- Concurrency: Kotlin Coroutines
- Navigation: Jetpack Navigation
